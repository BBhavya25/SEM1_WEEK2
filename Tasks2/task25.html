Exploring the Box Model

The box model consists of content, padding, border, and margin. Below are examples demonstrating how changing these properties affects the layout.

Box 1: Small margin, large padding, no border.
Box 2: Large margin, small padding, thin border.
Box 3: No margin, moderate padding, thick border.
Box 4: Large margin, no padding, dashed border.
Box 5: Equal margin and padding, double border.
